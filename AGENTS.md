# code-rag
code-rag is a stdio RAG server for an index of code repositories indexed to accelerate design, debugging, and discovery of relevant code prior to the use of filesystem tools. Use code-rag for vague or exploratory queries about a codebase; start with `code_rag_index_status` then discover relevant code using the search tool with natural language topic descriptions.
